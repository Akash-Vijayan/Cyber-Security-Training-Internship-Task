6. How Dmitry Works Internally
------------------------------------------------------------
- WHOIS lookup: Connects to WHOIS servers to get registration info.
- Netcraft: Queries Netcraft.com for domain details.
- Subdomain & email search: Uses web search and parsing.
- Port scan: Sends TCP SYN packets to check open ports.
- Banner grabbing: Reads service headers from open ports.

8. Detailed Flag Functions
------------------------------------------------------------
1. Domain WHOIS information (-w)
   - Domain registration details (registrar name, creation date, expiration date, owner info if public)
   - Name servers
   - Contact details (unless hidden)
   Use case: Know who registered the domain and when it expires.

2. IP WHOIS information (-i)
   - ISP or organization owning the IP address
   - Geographic location
   - IP block range
   Use case: Identify hosting provider.

3. Netcraft.com site information (-n)
   - Hosting history
   - Web server technology
   - SSL certificate details

4. Subdomain search (-s)
   - Finds possible subdomains

5. Email address search (-e)
   - Finds emails linked to the domain

6. TCP Port scan (-p)
   - Finds open ports and services

7. Filtered port reporting (-f)
   - Shows if ports are filtered by firewall

8. Banner grabbing (-b)
   - Gets service version info
